# Welcome to My First MDX Blog Post 🚀

Hello, my name is **Admin**, and this is my very first post written in **MDX** (Markdown + JSX).

## 📝 Why This Post Exists
This post is here to test:
- ✅ **Markdown rendering** (headings, lists, images, and code)
- ✅ **MDX support** (using components directly inside a post)
- ✅ **Layout styling** (to make sure everything looks consistent)

## 🔧 Tech Stack Behind This Blog
Here’s what powers this portfolio blog:
- **Next.js** – for server-side rendering and static generation
- **MDX** – for writing blog posts using Markdown with React components
- **Tailwind CSS** – for styling with utility-first classes
- **Vercel** – for seamless deployment and hosting
